>### 实验一 交互式SQL
导出整个数据库
```
mysqldump -u 用户名 -p 数据库名 > 导出的文件名
mysqldump -u dbuser -p dbname > dbname.sql
```
导入数据库
```$xslt
mysql -u root -p
mysql>use 数据库
mysql>source d:/dbname.sql
```
